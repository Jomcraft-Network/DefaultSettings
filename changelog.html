Latest release v4.0.6

  • Fixed: Improved keybinding detection to fix incompatibility with FTB Ultimine Addition

v4.0.5

  • Fixed NPE occuring on freshly setup modpacks
  • Fabric: Fixed bugged version checker, introduced in version 4.0.3
  • Forge: Removed unused legacy event hook, causing error messages appearing in versions beyond 1.12

  • Added automatic "creator check" to the /ds commands; Running them as the modpack's user (not the creator) will prompt you with a warning now.

v4.0.3

  • Added support for wildcard to the /ds saveconfigs command (thanks to ClaudiusMinimus)
  • The /ds saveconfigs command now supports whitespaces and quotes
  • Enhanced check if the current instance starts up the first time (thanks to ClaudiusMinimus and zaftnotameni)
  • DefaultSettings now prints more information, if the interaction with JCPlugin fails
  • Note: Fabric 1.16-1.19.3 and Forge 1.7.10-1.19.3 now share the same codebase! No more quirky differences!
